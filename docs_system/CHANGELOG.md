# Release Notes

## Studio AI - v0.4.0 (2026-01-16)

### Added
- **Activity History Enhancements**:
  - Image preview with Viewer.js integration (eye icon hover overlay)
  - Admin can view all users' activity history
  - Admin can delete activity entries including associated thumbnails
- **User Management**:
  - Manual credit adjustment field in Edit User page
  - Admin can directly set user's available credits
- **Registration Approval Flow**:
  - New users register as inactive by default (`is_active = false`)
  - Email notification sent to admins on new registration
  - Login blocked for inactive users with approval message
  - Admin account activation via Edit User page
- **UI Improvements**:
  - Register link added to login page footer
  - Credit balance badge in navigation bar (amber theme, responsive)

### Changed
- Login page layout updated with register link
- Navigation bar now displays user credits with coin icon
